# 肉鸽网页游戏 (Roguelike Web Game) - 规格说明书

## 1. 项目概述
- **项目名称**: 迷宫探险 (Dungeon Explorer)
- **类型**: 回合制肉鸽网页游戏
- **核心玩法**: 玩家在随机生成的迷宫中探索，击败敌人，收集装备，挑战越来越强的关卡
- **目标用户**: 休闲玩家

## 2. UI/UX 规格

### 布局结构
- **游戏画布**: 居中显示，800x600px
- **顶部状态栏**: 玩家血量、金币、当前层数
- **底部日志区**: 显示战斗和事件信息

### 视觉设计
- **主色调**: 
  - 背景: #1a1a2e (深紫黑)
  - 地面: #16213e (深蓝)
  - 墙壁: #0f3460 (深蓝)
  - 玩家: #e94560 (玫红)
  - 敌人: #ff6b35 (橙红)
  - 金币: #ffd700 (金色)
  - 生命: #ff4757 (红)
- **字体**: "Press Start 2P" (像素风), fallback: monospace
- **阴影**: 2px 黑色投影增加层次感

### 组件
- **玩家**: 🔴 红色圆点，带生命值显示
- **敌人**: 👹 随机表情符号 (👹, 👾, 💀, 👻, 🧟)
- **金币**: 🪙 金币图标
- **药水**: 🧪 恢复药水
- **武器**: ⚔️ 攻击装备
- **盾牌**: 🛡️ 防御装备

## 3. 功能规格

### 核心机制
1. **回合制移动**: 使用方向键或WASD移动，每步消耗1点能量
2. **战斗系统**: 靠近敌人自动进入战斗，伤害 = 攻击力 - 敌人防御
3. **升级系统**: 击败敌人获得经验，升级时恢复生命并提升属性
4. **装备系统**: 随机掉落武器、防具、药水
5. **随机生成**: 每次进入新层数生成新迷宫

### 游戏流程
1. 玩家从第1层开始
2. 在迷宫中探索，击败敌人
3. 找到楼梯下楼（每5层boss）
4. 死亡后显示最终层数和分数

### 属性系统
- **生命值**: 100 (最大)
- **攻击力**: 10
- **防御力**: 0
- **金币**: 0
- **经验**: 0

### 敌人类型
1. 🐀 老鼠 - 生命10, 攻击3, 防御0, 经验5, 金币5
2. 🦇 蝙蝠 - 生命8, 攻击5, 防御0, 经验8, 金币8
3. 🧟 僵尸 - 生命20, 攻击8, 防御2, 经验15, 金币15
4. 👹 恶魔 - 生命30, 攻击12, 防御5, 经验25, 金币25
5. 🐉 龙 (Boss) - 生命100, 攻击20, 防御10, 经验100, 金币100

### 物品掉落
- 🗡️ 攻击力+2-5
- 🛡️ 防御力+2-5
- 🧪 恢复药水 (生命+30)
- 💎 宝石 (额外金币)

## 4. 验收标准
- [x] 游戏可以正常加载运行
- [x] 玩家可以用方向键/WASD移动
- [x] 迷宫随机生成且不可通行
- [x] 敌人可以被击败
- [x] 击败敌人获得经验和金币
- [x] 可以拾取物品并生效
- [x] 可以下楼到下一层
- [x] 死亡时显示游戏结束
- [x] UI显示玩家状态
